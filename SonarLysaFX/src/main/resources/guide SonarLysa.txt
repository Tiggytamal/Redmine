1. Connexion

1.1 Connexion à l'appli
Permet de se connecter aux applications RTC et Sonar et d'activer les fonctions du Menu. Une fois les informations rentrées, il faut un peu de temps pour que la connexion soit validée. Le bouton passe alors à Deconnexion

1.2
Un clic sur le bouton deconnexion permet de fermer l'accès. Les menus sont alors désactivés.

2.Menu

2.1  Fonctions

2.1.1 Création Vue MEP/TEP
Permet de créer les vues Mensuelles et trimestrielles. Pour cela, il suffit de selectionner les dates voulues et de lancer le traitement.

2.1.2 Fichiers de Suivi
Lance le traitement des fichiers de suivi des anomalies datastage et java

2.1.3 Création vues maintenance
Création des vues pour les lots CHC et CHC_CDM

2.1.4 Mise à jour Lots RTC
Mise à jour du fichier de sauvegarde des WiLots RTC. Récupère tous les lots dans la date de création est supérieure à la date fournie (écrase la precédente suavegarde).

2.1.5 Création Autre Vues
Fénêtre des création des autres type de vues posisble dans Sonar.

3.1.5.1 Vue Datastage
Création de la vue de tous les composants DataStage

3.1.5.2 Vue Patrimoine
Création de la vue patrimoine à la semaine actuelle

3.1.5.3 Vues par applications
Suppression et recréation des vues par code applications

3.1.5.4 Vues par Edition
Création des vues par Edition (E31,E32,...) selon le paramétrage de l'application

2.2 Outils

2.2.1 Options
Permet la gestion des paramètres de l'application

2.2.1.1 Chargement fichiers
Permet le chargement et la sauvegarde dans le fichier fichier.xml des données :
- Applications : listes des applications enregistrées
- Referentiel Clarity : Liste de tous les projets Clarity
- Chef de Service : Liste de tous les chefs de Service connus
- Edition : Liste des éditions en fonctions du code version

2.2.1.2 Paramètres
Liste des paramètres de l'application (chaînes de caractères, booléens)

2.2.1.3 Autres paramètres
Liste des paramètres de l'application (Textes et liste de données)

2.2.1.4 Noms colonnes
Liste des noms des colonnes de chaque fichier excel de traitement ou de sortie.

2.2.2 Planificateur
Paramètrage et lancement du planificateur de tâches :
- Suivi Qualité : Traitement des fichiers de suivi (tous)
- Vues CHC : Création des vues CHC
- Vues CDM : création des vues CHC_CDM

2.2.3 Mise à jour vues
Lance la mise à jour des vues dans SonarQube. Celle-ci est faite automatiquement toutes les deux heures. Une confirmation est demandée avant lancement.

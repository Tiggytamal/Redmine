Gestion des fichiers de Suivi

Cette fonction permet de gérer les différents fichiers de suivi des anomalies :
  • Pour le JAVA : Suivi JAVA
  • Pour le DataStage : Suivi DataStage
  • Pour le COBOL : Suivi COBOL
  • Pour la triple analyse : Suivi Complet. Chaque matière est analysée en suivant. Puis un regroupement est fait pour gérer les anomalies multi-matières.



Attention, il ne faut pas que les fichiers excels soient déjà ouverts dans une autre application pendant l'analyse.
